## Config Fix Plan

### Host Settings
1. 현재 admin 페이지에서 등록된 host는 name, ip, user, password, ssh limit, modifier, actions 이렇게 7개 컬럼이 주어진다.
2. user와 password는 host별로 1개가 아니라, 여러개 등록 할 수 있게 변경하려고 한다.
3. Front에서 어떤식으로 표현하면 좋을지 고민해보고, 그에 맞게 등록할 수 있는 기능도 추가해야한다.

## RTD Settings

2. 적절하게 기능을 추가하라.
